Extended Bio

I was born in the early 2000s in Gujarat-India, where I still live.
Growing up, I had early access to a computer and the internet, so I spent a lot of time googling stuff.
I was always interested in how things were built: games, websites, robots (I watched a lot of sci-fi, lol).
That curiosity shaped the way I learn: if I find something interesting, I dig into it and figure it out.

I was also always drawn to art and design. I grew up drawing and painting.
When I got my first PC (a HP Pavilion idk the version) in middle school, I discovered MS Paint.
I started playing around with it and spent hours creating pixel art and cartoon characters.

My first actual introduction to programming was in middle school.
We had this “Intro to C Language” class when I was in 8th grade.
The class held about 2 times per week with only one day to access the computers, but it was enough to get me interested.
At this point, I didn’t even know much of what it was capable of. I just knew I enjoyed what was happening.

Later, we had another course in web development.
It was on the fundamentals of HTML and CSS.
This time, I actually somewhat understood how it worked and what it was capable of.
My favorite command I still remember it was <marquee />
From there, I started building static websites for fun.
I was also designing user interfaces (in MS Paint!) for the websites I was building.
I didn’t even know UI design was a thing back then.

So when it came time to pick a college major, Computer Science felt like the obvious choice.
And now, that’s what keeps me happy :)

About this Site

This site is my little corner of the internet. I’m constantly tweaking it to improve the experience.
I post nicheless notes on whatever catches my interest, and once in a while, I write longer pieces about software or design.

Writing is still relatively new to me, but I’ve found it helps me understand my own thoughts better.
If nothing else, it’s a record of what I was thinking about at any given time.

For a more technical breakdown of how I built this site, check out /colophon.
